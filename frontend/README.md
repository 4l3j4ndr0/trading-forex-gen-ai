# Forex Trading AI — Frontend

## Instalar dependencias

```bash
npm install
```

### Iniciar en modo desarrollo (HMR)

```bash
npx quasar dev
```

### Build para producción

```bash
npx quasar build
```

### Configuración

Ver [Configuring quasar.config.ts](https://quasar.dev/quasar-cli-vite/quasar-config-file).
